This is a sample PDF for testing digital signatures with PdfSigner.

\section*{How to Sign}

\begin{enumerate}
\item Drag this file onto \texttt{sign.bat}
\item Select your certificate from the popup
\item Enter your PIN if using a smart card
\item The signed file will be saved as \texttt{sample\_signed.pdf}
\end{enumerate}

\section*{Command Line}

\begin{verbatim}
.\PdfSigner.exe sample.pdf --gui
.\PdfSigner.exe --verify sample_signed.pdf
\end{verbatim}
